const Esc_UINT8  cipher[],
Esc_UINT8  plain[] 
)


Decrypts one 3-DES block in ECB mode.

Performs the TDEA encryption operation:
I - DES_D_K3 - DES_E_K2 - DES_D_K1 - O

Block size is EscDes_BLOCK_BYTES. The context must be initialized with EscDes3Ecb_Init.

The parameters plain and cipher can point to the same location.

Parameters
[in]ecbCtxContext initialized with EscDes3Ecb_Init.
[in]cipherCiphertext to decrypt. Must be exactly EscDes_BLOCK_BYTES bytes.
[out]plainDecrypted ciphertext. Must be exactly EscDes_BLOCK_BYTES bytes.
Returns
Esc_NO_ERROR if everything works fine.

Initializes the ECB Context with the corresponding 3-DES key.

This function does not check the key for weaknesses nor does it check the parity bits.

To check if a key is a known weak key, use the function EscDesKeytest_IsKeyWeak().
