this function generates a key according to the NIST 800 56Ar2 key derivation function using SHA256

Note
: max size of input data (zLength + otherInfoLength) cannot exceed max of 32-bit unsigned int, 2^32 -1
Parameters
[out]key: pointer to byte array to hold key generated by this function
[in]keySize: Size in bytes of the key to be generated
[in]z: pointer to shared secret data byte array
[in]zLength: length in bytes of z array
[in]otherInfo: pointer to byte array of additional info for key derivation
[in]otherInfoLength: size in bytes of the otherInfo byte array
Returns
0 Esc_NO_ERROR on success, !0 on failure
