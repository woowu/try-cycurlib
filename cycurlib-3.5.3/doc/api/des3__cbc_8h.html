Decrypts one or more data blocks and updates the DES context.

The context must be initialized with EscDes3Cbc_Init. Plain pointer can be the same as cipher pointer.

Parameters
[in]cbcCtxThe previous DES CBC context.
[out]cbcCtxThe updated DES CBC context.
[in]cipherCipher data bytes array.
[out]plainWill contain the plaintext after the function call.
[in]lengthLength of plain and cipher data in bytes. Has to be multiple of EscDes_BLOCK_BYTES bytes.
Returns
Esc_NO_ERROR if everything works fine.

Initializes the CBC Context with the corresponding DES key and IV.

This function does not check the key for weaknesses nor does it check the parity bits.

To check if a key is a known weak key, use the function EscDesKeytest_IsKeyWeak().

See also
des_keytest.h
Parameters
[out]cbcCtxThe DES CBC context.
[in]key1TDEA Key1. Key bytes array. Array of size EscDes_KEY_BYTES.
[in]key2TDEA Key2. Key bytes array. Array of size EscDes_KEY_BYTES.
[in]key3TDEA Key3. Key bytes array. Array of size EscDes_KEY_BYTES.
[in]iv64-bit initialization vector. Array of EscDes_BLOCK_BYTES bytes.
Returns
Esc_NO_ERROR if everything works fine.
