typedef Esc_ERROR(* EscRandom_GetRandom) (void *prngState, Esc_UINT8 rnd[], const Esc_UINT32 len)

Function prototype for pseudo-random number generators (PRNGs).

Various functions in the CycurLIB expect such a PRNG to generate random numbers. One implementation of this prototype is the Hash_DRBG contained in the CycurLIB. See EscHashDrbg_GetRandomWrapper() in file hash_drbg.h.

The PRNG is expected to be cryptographically secure! Implementations such as rand() from the C standard library are NOT secure and should not be used in this context!

Parameters
[in]prngStateRandom state as your random implementation needs it.
[out]rndPointer to the array where the random data will be stored
[in]lenRequested length of random data
Returns
Esc_NO_ERROR if everything works fine
