Detailed Description

PKCS#1 RSAES-v1_5 encryption and decryption.

Byte format of all long numbers is Big-Endian.

See also
www.emc.com/collateral/white-papers/h11300-pkcs-1v2-2-rsa-cryptography-standard-wp.pdf
Rev
3832

#define EscPkcs1RsaEsV15_MAX_MESSAGE_LENGTH   (EscRsa_KEY_BYTES_MAX - EscPkcs1RsaEsV15_MIN_PADDING_LENGTH)

The maximum message given the largest configured RSA key size.

Depending on the used key, the actual maximum message size may be smaller.

Esc_ERROR EscPkcs1RsaEsV15_DecryptRun ( EscPkcs1RsaEsV15_DecryptContext *  ctx)


Run decryption.

Each call may only compute a small part of the overall computation. Consequently, this function needs to be called until it returns an error code different from Esc_AGAIN.

Parameters
[in]ctxThe context structure which stores the state of the decryption.
Return values
Esc_AGAINif this function needs to be called again to continue decryption.
Esc_NO_ERRORif decryption finished successfully. The ciphertext was written to the buffer passed to EscPkcs1RsaEsV15_DecryptInit().
Esc_DECRYPTION_FAILEDThe ciphertext could not be decrypted correctly.
