Detailed Description

RIPEMD-160 implementation.

According to: ISO/IEC FDIS 10118-3 Final Draft 2003, Chapter 7 algorithm from:

  • Hans Dobbertin, Antoon Bosselaers, Bart Preneel:
  • "RIPEMD-160:A Strengthened Version of RIPEMD"

Esc_ERROR EscRipemd160_Calc ( const Esc_UINT8  message[],
Esc_UINT32  messageLength,
Esc_UINT8  digest[],
Esc_UINT8  digestLength 
)


This function calculates and returns a RIPEMD-160 hash

Parameters
[in]messagePointer to input data. May be a NULL pointer if messageLength is 0.
[in]messageLengthLength in number of bytes of the message.
[out]digestBuffer to store digest value of digestLength bytes. Can be the same as the message buffer.
[in]digestLengthThe desired length of the digest to be stored in the buffer "digest". This parameter can be used to truncate the digest such that only the first bytes are copied to the output buffer. The value must not be zero and must not exceed EscRipemd160_DIGEST_LEN.
Returns
Esc_NO_ERROR if everything works fine.
