Esc_ERROR EscBfishEcb_Decrypt ( const EscBfishEcb_ContextT *  ecbCtx,
const Esc_UINT8  cipher[],
Esc_UINT8  plain[] 
)


Decrypts one Blowfish block in ECB mode.

Block size is EscBfish_BLOCK_BYTES. The context must be initialized with EscBfishEcb_Init.

The parameters plain and cipher can point to the same location.

Parameters
[in]ecbCtxContext initialized with EscBfishEcb_Init.
[in]cipherCiphertext to decrypt. Must be exactly EscBfish_BLOCK_BYTES bytes.
[out]plainDecrypted ciphertext. Must be exactly EscBfish_BLOCK_BYTES bytes.
Returns
Esc_NO_ERROR if everything works fine.
