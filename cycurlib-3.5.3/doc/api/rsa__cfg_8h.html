◆ EscRsa_PRIMETEST_ITER

#define EscRsa_PRIMETEST_ITER   5U

Size of max.

RSA modulus (in bit): 1024, 1536, 2048, 3072, 4096 Number of iterations of the Miller-Rabin primality test. Only used during key generation.

◆ EscRsa_USE_MONTGOM_MUL

#define EscRsa_USE_MONTGOM_MUL   1

Enable the Montgomery Reduction.

Increases the speed unless the VERY short exponents (3) are used. This option affects the exponentiation for both long exponents and short exponents ( ModExpPub() / ModExpPriv() ).

◆ EscRsa_USE_SLIDING_WINDOW

#define EscRsa_USE_SLIDING_WINDOW   0

Enable for using the Sliding Window algorithm Increases speed, but also increases code size and stack used.

This option affects the exponentiation for private (long) exponents ( ModExpPriv() ).

◆ EscRsa_WINDOW_SIZE

#define EscRsa_WINDOW_SIZE   4U

Window size for the Sliding Window algorithm Only used if the sliding window algorithm is enabled (EscRsa_USE_SLIDING_WINDOW == 1).

The higher the window size, the higher the RAM usage!
