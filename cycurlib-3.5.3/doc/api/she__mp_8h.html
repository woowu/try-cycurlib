◆ EscSheMpComp()

Esc_ERROR EscSheMpComp ( const Esc_UINT8  message[],
const Esc_UINT32  bitLength,
Esc_UINT8  digest[16] 
)

This function calculates the compression function as specified in the SHE functional specification using the Miyaguchi Preneel construction.

The input message will be padded and split into blocks automatically.

Parameters
[in]messageThe message to be compressed.
[in]bitLengthThe size of the message in bits, does not need to be aligned to full bytes.
[out]digestThe final output of the compression function.
Returns
Esc_NO_ERROR if everything is fine.

Esc_UINT8  derived[16]

This function calculates the key derivation as specified in the SHE functional specification as KDF(K, C) = AES-MP(K | C).

Parameters
[in]keyThe input key for the key derivation function. Has to be 128 bit / 16 bytes in size.
[in]typeThe type of input key as one of the constants.
[out]derivedThe derived key as a 16 byte array.
Returns
Esc_NO_ERROR if everything is fine.
