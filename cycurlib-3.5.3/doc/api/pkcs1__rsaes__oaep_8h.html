Detailed Description

PKCS#1 v2.2 RSAES-OAEP encryption and decryption with MGF1 as the mask generation function.

Byte format of all long numbers is Big-Endian.

See also
www.emc.com/collateral/white-papers/h11300-pkcs-1v2-2-rsa-cryptography-standard-wp.pdf

3832

#define EscPkcs1RsaEsOaep_MAX_MESSAGE_LENGTH   (EscRsa_KEY_BYTES_MAX - (2U * EscPkcs1RsaEsOaep_MIN_DIGEST_LEN) - 2U)

The maximum message given the largest configured RSA key size and the smallest digest length.

Depending on the used key and digest type, the actual maximum message size may be smaller.

Decrypts a message according to RSAEP-OAEP.

See also
Section 7.1.2
Parameters
[in]cipherCiphertext to be decrypted.
[in]cipherLenLength of the ciphertext in bytes. This must match the modulus size of the given private key (e.g. for RSA-1024, a length of 1024 / 8 = 128 bytes).
[in]labelLabel to be verified. This parameter is optional and may be set to NULL.
[in]labelLenSize of the label in bytes. Must be 0 if label is set to NULL.
[out]messageDecrypted message.
[in,out]messageLenIn: Size of the message buffer in bytes. EscPkcs1RsaEsOaep_MAX_MESSAGE_LENGTH can be used as the maximum size allowed by configuration. Out: Size of the message in bytes
[in]privKeyPrivate key structure.
[in]digestTypeThe type of the digest. One of the EscPkcs1_DIGEST_TYPE_XXX defines from pkcs1.h.
Returns
Esc_NO_ERROR if everything works fine or Esc_DECRYPTION_ERROR if decryption fails.

Esc_ERROR EscPkcs1RsaEsOaep_DecryptRun ( EscPkcs1RsaEsOaep_DecryptContext *  ctx)


Run decryption.

Each call may only compute a small part of the overall computation. Consequently, this function needs to be called until it returns an error code different from Esc_AGAIN.

Parameters
[in]ctxThe context structure which stores the state of the decryption.
Return values
Esc_AGAINif this function needs to be called again to continue decryption.
Esc_NO_ERRORif decryption finished successfully. The ciphertext was written to the buffer passed to EscPkcs1RsaEsOaep_DecryptInit().
Esc_DECRYPTION_FAILEDThe ciphertext could not be decrypted correctly.
