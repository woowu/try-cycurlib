AES key unwrap key function, takes encrypted key and provided clear text of key

Parameters
[in]kekkey encryption key
[in]kekLenlength of key encryption key in bits
[in]ivinitialization vector, can be null and default of a6 will be used; 8 bytes
[in]inwrapped key to be decrypted
[out]outstorage location for unwrapped key; 8 bytes smaller than input
[in]lenlength of input wrapped key in bytes. Must be at least 24 bytes and must be a multiple of 8.
Returns
Esc_NO_ERROR on success
