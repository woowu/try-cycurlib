Detailed Description

AES implementation (FIPS-197 compliant) Common functions. The functions are not supposed to be called by the user. Use one of the submodule functions instead.

See also
FIPS-197

Key sizes of 128, 192 and 256 bits are allowed.

Rev
4191

Decrypts one AES block.

Block size is EscAes_BLOCK_BYTES. The context must be initialized with EscAes_Init.

The parameters plain and cipher can point to the same location.

Parameters
[in]ctxContext initialized with EscAes_Init.
[in]cipherCiphertext to decrypt. Must be exactly EscAes_BLOCK_BYTES bytes.
[out]plainDecrypted ciphertext. Must be exactly EscAes_BLOCK_BYTES bytes.
Returns
Esc_NO_ERROR if everything works fine.
