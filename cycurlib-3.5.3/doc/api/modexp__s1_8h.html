Initialize modular exponentiation for private (large) exponents.

Modular exponentiation means: y = b^e mod m.

Parameters
[in]ctxThe context to store state and intermediate results of the computation.
[in]exponentThe exponent e.
[in]keySizeBitsThe size of the modulus in bits.
[in]modulusThe modulus m.
[in,out]inOutIn: The base b of the modular exponentiation. Out: The result y of the modular exponentiation.

◆ EscModExpS1_PrivRun()

Esc_ERROR EscModExpS1_PrivRun ( EscModExpS1_PrivContext *  ctx)


Run modular exponentiation for private (large) exponents.

The context must have been initialized previously by calling EscModExpS1_PrivInit(). This variant is sliced which means that the function may need to be called multiple times until a return code different from Esc_AGAIN is returned.

Parameters
[in]ctxThe context to store state and intermediate results of the computation.
Returns
Any of the below return codes on success or a corresponding error code on failure.
Return values
Esc_AGAINThe computation is not finished and the function must be called again.
Esc_NO_ERRORThe computation finished successfully. The result is stored in the inOut field element passed to EscModExpS1_PrivInit().
