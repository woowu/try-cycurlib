Decrypts one data unit (i.e., sector).

The context must be initialized with EscAesXts_Init.

Parameters
[in]ctxinitialized AES XTS context.
[in]cipherCiphertext to decrypt.
[out]plainArray to write plaintext to encrypt.
[in]lengthLength of plain and cipher data in bytes. Has to be at least EscAes_BLOCK_BYTES bytes.
[in]tweaktweak value. Array of size EscAes_BLOCK_BYTES.
Returns
Esc_NO_ERROR if everything works fine.

Initializes the AES-XTS Context with the corresponding AES keys.

Parameters
[out]ctxThe AES XTS context.
[in]keyBitsBit length of the AES key. Can be 128 or 256.
[in]key1First key byte array. The size of the key depends on the keyBits parameter: 128 bits => 16 bytes 256 bits => 32 bytes
[in]key2Second key byte array. The size of the key depends on the keyBits parameter: 128 bits => 16 bytes 256 bits => 32 bytes
Returns
Esc_NO_ERROR if everything works fine.
